## 分析

### 1. 字符串操作
将整数转成字符串，进行字符串反转后，进行判断并返回。对于溢出的判断暂时没有好的想法，只能借助于异常

### 2. 数学计算
循环依次将原数据`mod`10得到末尾数字，累加到返回的变量n。下一次循环时n*10+mod
```Java
    public static int reverse3(int x) {
        int r = 0;
        while (x != 0) {
            //暂存当前值，后续比较使用
            int tmp = r;
            //先进位再取mod
            r = r * 10 + x % 10;
            //舍去末尾
            x /= 10;
            //一旦溢出之后，将不再与原数相等。证明？
            if (r / 10 != tmp)
                return 0;
        }
        return r;
    }
```
